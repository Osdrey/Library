# Software de Biblioteca Universitaria

## Descripción
Este software de biblioteca estará diseñado para la gestión eficiente de préstamos y reservas de material bibliográfico en una universidad. Su objetivo es facilitar el acceso y control de los recursos de la biblioteca para diferentes tipos de usuarios: estudiantes, profesores y empleados. 

## Funcionalidades

### Funciones de los usuarios
- Registrarse en la plataforma y acceder mediante un sistema de autenticación.
- Consultar el catálogo de materiales disponibles.
- Realizar reservas de material bibliográfico.
- Ver el historial de reservas realizadas.
- Consultar los préstamos activos.

### Funciones del bibliotecario
- Visualizar las reservas pendientes.
- Aceptar o rechazar solicitudes de reserva de material.
- Cuando una reserva es aceptada, se genera automáticamente un préstamo asociado.

### Funciones del administrador
- Gestionar materiales, préstamos, reservas y usuarios.

## Alcance actual del software
El software se encuentra en fase beta. Se encuentra en desarrollo.
